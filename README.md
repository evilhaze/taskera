# Taskera

Taskera — современная SaaS-платформа для управления задачами и проектами.
Сервис помогает командам планировать работу, отслеживать прогресс задач и эффективно организовывать рабочие процессы.

Платформа вдохновлена такими сервисами как ClickUp, Trello и Asana, но делает упор на простоту, скорость и современный интерфейс.

---

## Основные возможности

* управление проектами
* канбан-доска задач
* создание подзадач (subtasks)
* назначение исполнителей
* приоритеты задач
* теги и метки
* комментарии к задачам
* аналитика задач на дашборде
* система активности команды
* drag & drop перемещение задач
* AI-ассистент для генерации задач

## Интерфейс системы

Платформа включает:

* Dashboard аналитики
* Kanban board задач
* страницу проектов
* управление участниками команды
* систему уведомлений
* AI Assistant

## Запуск проекта локально

Установить зависимости:

npm install

Запустить проект:

npm run dev

После запуска приложение будет доступно по адресу:

http://localhost:3000
